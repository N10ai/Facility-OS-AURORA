# FacilityOS Operations Core

Advanced feature development is frozen until the core cleaning workflow is production-ready.

## Current delivery sequence

1. Finance lifecycle: quotes, invoices, partial payments, expenses, employee pay.
2. Facility mapping: floors, zones, areas and area metadata.
3. SOP engine: procedures, checklists, supplies, time standards and proof rules.
4. Guided work orders: phone-first execution by area.
5. Mobile photos: direct camera capture, compression and before/after metadata.
6. Inspections, corrections and reporting.

## Definition of done

Each increment must be database-backed, mobile-friendly, connected to related records, validated by the production build and usable in real operations.
