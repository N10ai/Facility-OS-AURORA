# Release Notes — Aurora Operations v2.1.0

This release changes FacilityOS from a design prototype into a usable operating MVP.

- Admin, Employee, and Customer portals
- MIP starter profile
- Recurring annual scheduling
- Mission execution
- Supabase Storage proof uploads
- Verification gate
- Issue reporting
- Customer requests
